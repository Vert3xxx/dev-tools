# DevTools Hub  
A fully static, modern all-in-one developer toolkit.

## ✔ Tools Included
- JSON Formatter
- Base64 Encoder/Decoder
- Markdown Previewer
- UUID Generator
- Regex Tester
- Color Picker
- Image Compressor
- Hash Generator (SHA-256 + MD5)
- Timestamp Converter

## ✔ UI Features
- Sidebar navigation
- Light/Dark mode toggle
- Tool icons (no logo)
- Glass + neon modern theme
- Smooth scrolling

## ✔ Tech
- HTML
- CSS
- JavaScript
